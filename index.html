Plain files

Your notes stay as `.md` files in a repository you control, instead of disappearing into a proprietary workspace.

Static app

No backend service. GitNotes reads and writes through the GitHub Contents API using a token stored in this browser.

Offline drafts

Local browser storage keeps drafts available when you lose connection, then Push sends pending files back to GitHub.

Setup

Connect a repository once.

  1. Create a fine-grained GitHub token with repository Contents read/write access.
  2. Enter the owner, repo name, branch, and optional root directory.
  3. Start editing Markdown notes from the browser or installed PWA.
